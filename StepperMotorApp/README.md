# 28BYJ-48 Stepper Motor with ULN2003

### Opis

Test silnika krokowego 28BYJ-48 sterowanego przez układ ULN2003 przy użyciu Raspberry Pi i .NET 9.

### Połączenia

| ULN2003 | Raspberry Pi (BCM) |
| :------ | :----------------- |
| IN1     | GPIO 17            |
| IN2     | GPIO 18            |
| IN3     | GPIO 27            |
| IN4     | GPIO 22            |
| VCC     | 5V                 |
| GND     | GND                |
